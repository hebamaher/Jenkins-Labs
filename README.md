# Jenkins CI/CD Lab 1

## Overview
This lab demonstrates how to install Jenkins using Docker, configure role-based authorization, connect Jenkins with a private Git repository, and create different types of Jenkins pipelines including multibranch pipelines.

---

# Objectives
- Install Jenkins using Docker
- Configure Jenkins security and roles
- Create Jenkins users
- Configure Role-Based Authorization
- Create Freestyle jobs
- Create Declarative Pipelines
- Create Scripted Pipelines
- Use Jenkinsfiles inside Git repositories
- Configure Multibranch Pipelines

---

# Technologies Used

- Docker
- Jenkins
- Git
- GitHub
- Groovy
- Jenkins Pipeline

---

# Prerequisites

Before starting, make sure you have:

- Docker installed
- Git installed
- GitHub account
- Internet connection

# Jenkins Advanced CI/CD Lab 2

## Overview

This lab focuses on advanced Jenkins CI/CD operations including:

- Running Docker commands from Jenkins
- Building CI/CD pipelines
- Configuring Jenkins agents (slaves) on AWS EC2
- Integrating Slack notifications
- Using Jenkins audit logging
- Building and pushing Docker images using GitHub Actions
- Running Terraform through Jenkins pipelines

---

# Objectives

By the end of this lab, you will be able to:

- Configure Jenkins with Docker socket access
- Build CI/CD pipelines for Node.js applications
- Configure AWS EC2 instances as Jenkins agents
- Integrate Slack with Jenkins
- Send pipeline notifications to Slack
- Monitor Jenkins actions using audit logs
- Build Docker images using GitHub Actions
- Push Docker images to DockerHub
- Execute Terraform infrastructure pipelines using Jenkins

---

# Technologies Used

- Jenkins
- Docker
- GitHub
- GitHub Actions
- AWS EC2
- Terraform
- Slack
- Node.js
- Django
- DockerHub
